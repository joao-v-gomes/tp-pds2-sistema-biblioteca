Detailed Description

Armazena as caracteristicas do Bibliotecário.

O Bibliotecario é o administrador do sistema. Ele pode realizar todas as pesquisas que Cliente faz, porém, o Bibliotecário pode cadastrar itens no sistema, como por exemplo, livros e outros Clientes.

◆ pesquisarReserva()

std::list<Cliente> Bibliotecario::pesquisarReserva ( Livro  livro)

Realiza a pesquisa no banco de dados sobre as reservas de um determinado livro e retorna a lista uma lista de clientes.

A lista de reserva funciona como uma fila, o primeiro cliente é, de fato, com quem está o livro. O segundo cliente pegará o livro assim que o primeiro devolver o livro para a biblioteca.
